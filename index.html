소개

안녕하세요. UI 개발자로서 사용자 경험과 웹 표준을 고려한 화면 구현에 관심이 있습니다. 이 공간은 저의 역량과 학습 경험을 보여주기 위한 포트폴리오 사이트입니다.

기술 스택

  • HTML5 / CSS3 – 마크업 및 스타일링
  • JavaScript – 이벤트 처리 및 동적 UI 구현 (학습 진행 중)
  • 반응형 웹 – 다양한 해상도와 디바이스 대응 경험

작업 샘플

단순한 샘플용 화면이 아닌, 사용자 흐름과 콘텐츠 전달을 고려한 UI를 설계했습니다. 구조화된 마크업과 일관된 디자인을 통해 사용성과 심미성을 고려했습니다.
